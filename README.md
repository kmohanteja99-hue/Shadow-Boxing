Kinesis AI - Shadow Boxing 🥊
An interactive, AI-powered shadow boxing game that runs directly in your web browser. Using MediaPipe Pose for real-time body tracking, this project allows users to hit virtual targets and track their punch velocity using only their webcam.

🚀 Features
Real-time AI Tracking: Uses high-accuracy pose estimation to track hand movements.

Dynamic Motion Trails: Visualizes movement with glowing blue neon trails.

Impact & Velocity HUD: Calculates the "MPH" of your punches and tracks your total score.

Audio Feedback: Real-time sound effects triggered by target impacts.

No Extra Hardware: Runs on any device with a standard webcam.

🛠️ Built With
HTML5 / CSS3 - For the structure and neon-style HUD.

JavaScript (Vanilla) - Game logic and physics engine.

MediaPipe Pose - Google's state-of-the-art ML model for body landmark detection.

Web Camera API - For real-time video processing.

🎮 How to Play
Open the Game: Launch index.html in a modern browser (Chrome/Edge recommended).

Grant Permissions: Allow the browser to access your camera.

Position Yourself: Stand back until your upper body is visible in the frame.

Punch the Targets: Strike the orange orbs that appear on screen.

Listen: Click anywhere on the screen once to enable the hit sound effects.
